랫킨 생성기 사용 설명서 (Walkthrough) - v3
피드백을 반영하여 더욱 귀엽고 자연스러워진 랫킨 생성기입니다!

실행 방법
…/scratch/ratkin-world/index.html
 파일을 브라우저에서 엽니다.
v6.1 업데이트 내용
크기 조절 개선:
슬라이더 옆에 현재 배율(예: 2.0x)이 표시됩니다.
랫킨을 키우면 말풍선도 같이 커집니다.
인형뽑기 모드 개선 (꾹 눌러서 잡기):
클릭: 랫킨이 말을 합니다.
0.5초 꾹 누르기: 랫킨을 들어 올립니다(매달리기). 실수로 잡는 것을 방지합니다.
v5 업데이트 내용
상태별 대사: 랫킨들이 현재 행동(걷기, 뛰기, 대기)에 맞는 대사를 합니다. 서 있을 때만 "둠칫둠칫" 거려요!
다크 모드: 눈이 편안한 어두운 회색 배경이 기본으로 적용되었습니다.
v4 업데이트 내용
대기 상태 (Idle): 랫킨들이 가끔 멈춰서 두리번거립니다 (10% 확률).
배경색 커스텀: 원하는 색상으로 배경과 월드 색상을 꾸밀 수 있습니다. 패턴 색상도 자동으로 어울리게 변합니다!
v3 업데이트 내용
모바일 최적화: 스마트폰에서도 예쁘게 보이도록 화면 크기에 맞춰 레이아웃이 자동 조정됩니다.
대각선 이동: 랫킨들이 이제 상하좌우 자유롭게 돌아다닙니다.
충돌 감지: 랫킨끼리 부딪히면 콩! 하고 튕겨 나갑니다.
말풍선 개선: 말풍선이 화면 밖으로 잘리지 않도록 자동으로 위치를 조정합니다.
이미지 복구: 랫킨들이 다시 정상적으로 보입니다.
v2 업데이트 내용
자연스러운 움직임: 랫킨이 걷는 방향을 올바르게 바라보며, 다리를 움직이는 애니메이션이 추가되었습니다.
말풍선 개선: 랫킨이 뒤를 돌아도 말풍선은 뒤집히지 않고 올바르게 보입니다.
자동 대사: 클릭하지 않아도 가끔씩 랫킨들이 혼잣말을 합니다.
눈이 편한 배경: 배경색이 부드러운 연두빛으로 변경되어 오래 봐도 눈이 아프지 않습니다.
기능 설명
생성하기 (Generate): 버튼을 누르면 새로운 랫킨이 "뿅" 하고 나타납니다.
움직임 (Movement):
3초마다 행동(걷기/뛰기)을 결정합니다.
70% 확률로 걷고(Walk), 30% 확률로 뜁니다(Run).
상호작용 (Interaction):
랫킨을 클릭하면 대사를 합니다.
가만히 있어도 가끔씩 대사를 합니다.
초기화 (Reset): 모든 랫킨을 삭제합니다.
